Programmatic Access (MCP)

WEO data is accessible via the Model Context Protocol for programmatic query of verified Coordination Intelligence.

Connection endpoint: warmthengine.com/mcp

Available tools:

  • search_events — query by tier, bloc, domain, type, industry tag, or keyword
  • get_event — retrieve a single event with full assessment data
  • get_connections — coordination connections filtered by event, type, or confidence
  • get_connection_by_id — retrieve a single coordination connection
  • get_topology — coordination topology: whole-graph network statistics, or per-node metrics for a single event
  • traverse_coordination — walk the coordination lineage: ancestors, descendants, neighbours, or paths between two events
  • get_event_stack — the stack journey for one event: its coordination, material, and capability links at depth 1
  • get_threads — Infrastructure Threads: the factual material layer linking events that share a named AI hardware asset
  • get_actors — sovereign capability profiles by designation
  • query_scp — pivot and compare the SCP register: one dimension across all actors, actors side by side, or the Dimension Watch register
  • get_capability_links — event↔dimension membership links: which events qualify or contribute to each actor's SCP dimensions
  • get_capability_signatures — precomputed Capability Signatures: directional CC chains/hubs whose events span two or more SCP dimensions
  • get_blocs — the WEO Bloc Membership Register: each nation's bloc assignment and universal membership category
  • get_coverage_stats — corpus statistics and distribution overview
  • get_methodology — methodology context for any WEO-specific term or field

Full analytical access is by subscription. Obtain an access token from the Subscribe button above, then send it as an Authorization: Bearer weo_... header on the /mcp or REST endpoints - a ?token=weo_... query parameter also works. One token unlocks the full tier across MCP, the API, and the website.

Full tool reference (descriptions and input schemas): warmthengine.com/mcp
